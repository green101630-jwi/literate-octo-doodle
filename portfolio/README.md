# Portfolio Site Draft

This directory contains a simple static portfolio site for GitHub Pages.

## Files

- `index.html`: portfolio homepage
- `styles.css`: visual styling

## Suggested GitHub Pages setup

1. Go to repository settings.
2. Open **Pages**.
3. Set source to the `main` branch.
4. Use the root directory if the repository serves the whole site.
5. Open `/portfolio/` after deployment.
