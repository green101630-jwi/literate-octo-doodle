Featured Project

OMX Pick-and-Place Robot Learning

In Progress

This project focuses on building a robot learning pipeline for OpenManipulator-X based pick-and-place tasks. The current work covers demonstration collection, dataset formatting, behavior cloning experiments, IsaacSim/IsaacLab validation, and sim-real hardware mapping.

01

Demonstration Collection

Teleoperation and script-based demonstrations for pick-and-place style manipulation tasks.

02

Dataset Formatting

Joint trajectories and demonstrations organized into HDF5/NPZ and robomimic-style formats.

03

Behavior Cloning

BC training candidates prepared using collected demonstrations and manipulation task observations.

04

Sim-Real Mapping

Analysis of IsaacSim/IsaacLab contact behavior and real-to-sim gripper representation differences.

My Role

  • Built ROS2-based joint trajectory recording and replay structure.
  • Organized teleoperation demonstrations for BC training candidates.
  • Analyzed grasp instability differences between IsaacSim and IsaacLab.
  • Investigated 1-joint real gripper command versus 2-joint simulation gripper representation.
  • Prepared the software pipeline for later real-hardware deployment.
